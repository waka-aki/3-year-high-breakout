# 52週高値ブレイクアウトスクリーナー

東証プライム・スタンダード・グロース市場の日本株を対象とした52週高値ブレイクアウトスクリーナー。
ブレイクアウトを検出し、出来高・ボラティリティ指標でスコアリングし、ブレイクアウト後のパフォーマンスを追跡し、HTMLダッシュボードを出力する。

## セットアップ

```bash
python -m venv 52_breakout
source 52_breakout/bin/activate
pip install -r requirements.txt
```

## 実行方法

```bash
source 52_breakout/bin/activate
python src/main.py --update-tickers   # 初回: 銘柄リストDL + 株価取得
python src/main.py                     # 2回目以降: キャッシュされた銘柄を使用
```

## 銘柄の抽出条件

ダッシュボードに掲載される銘柄は、以下の条件をすべて満たしたものです。

- **対象市場**: 東証プライム・スタンダード・グロース市場の全内国株式（ETF・REIT等は除外）
- **ブレイクアウト判定**: 本日の終値が過去252営業日（約1年）の高値を上回った銘柄
- **新規のみ**: 前日時点では52週高値を超えていなかった銘柄だけを検出します。連日ブレイクアウト中の銘柄は除外されます
- **30日クールダウン**: 過去30日以内に一度ブレイクアウトとして検出された銘柄は、重複して表示されません
- **最低履歴要件**: 価格データが20営業日未満の銘柄（新規上場直後など）は対象外です

## ダッシュボードの見方

### 1. Summary（サマリー）

ページ上部に表示される概要情報です。

- 本日のブレイクアウト銘柄数
- 市場別の内訳（Prime / Standard / Growth）
- ブレイクアウト銘柄の平均5日・30日リターン

### 2. Today's Breakouts（本日のブレイクアウト銘柄）テーブル

本日新たに52週高値を更新した銘柄の一覧です。各カラムの意味は以下のとおりです。

| カラム | 説明 |
|--------|------|
| Ticker | 銘柄コード |
| Name | 銘柄名 |
| Market | 市場区分（Prime / Standard / Growth） |
| Close | 本日の終値（円） |
| 52W High | 過去252営業日の高値（円） |
| Breakout % | 52週高値をどれだけ上回ったか（%）。大きいほど力強いブレイクアウト |
| Value Change | 本日の売買代金 ÷ 過去20日平均売買代金。1.0より大きいほど普段よりお金が集まっている |
| Vol-Adj Ratio | 出来高変化率 ÷ ボラティリティ（過去20日リターンの標準偏差）。値が高いほど、普段の値動きに対して異常に大きな出来高が入っていることを示す |
| Vol Days | 直近5日間で出来高が20日平均を超えた日数（0〜5）。★マークは3日以上で、出来高の増加が一時的ではなく継続的であることを示す |
| Rev Growth | 売上高の前年比成長率（%）。N/Aは決算データが未取得の銘柄 |
| Op Margin | 営業利益率（%）。N/Aは決算データが未取得の銘柄 |

### 3. Post-Breakout Performance（ブレイクアウト後パフォーマンス）テーブル

過去30日間にブレイクアウトした銘柄の、その後の値動きを追跡した一覧です。

- **5d / 30d / 60d / 90d**: ブレイクアウト日の終値を基準とした騰落率（%）。緑色はプラス、赤色はマイナスを示します
- まだその日数が経過していない期間は「-」と表示されます

## ダッシュボードの閲覧方法

GitHub Actions により毎日 JST 18:00 にスキャンが自動実行され、結果がリポジトリにコミットされます。

ダッシュボードを閲覧するには、リポジトリをローカルに取得してブラウザで開いてください。

```bash
# 初回
git clone <リポジトリURL>
# ブラウザで開く
open output/dashboard.html        # macOS
xdg-open output/dashboard.html   # Linux
start output/dashboard.html      # Windows

# 2回目以降（最新結果を取得）
git pull
```

## 自動実行 (GitHub Actions)

- **スケジュール**: 毎日 JST 18:00 (UTC 09:00)
- **手動実行**: GitHub リポジトリの Actions タブから `workflow_dispatch` で手動実行も可能
- **対象ファイル**: `data/*.csv`, `output/dashboard.html`, `logs/` が自動コミットされる
